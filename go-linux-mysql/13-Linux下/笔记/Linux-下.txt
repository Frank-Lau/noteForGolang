find：--- 按文件属性  【重点】

	语法： find  搜索目录位置  参数  关键字。

	-name：按名字搜索
		find ./ -name "init*"

	-type:   按类型搜索
		find ./ -type "p"/"s"/"d"....

		普通文件： “f”

	-size： 按大小搜索	
		find ./ -size +3k -size -5k   （第2个-size不能省略）

		+：大于、-：小于

		k ：必须小写！

		M：必须大写！

		无单位：按扇区（512B）个数计算

	-maxdepth： 指定搜索层级

		指定搜索层级深度。应该防置于其他参数之前。

		find ./ -maxdepth 1 -type "d" -name "go*"

	扩展自学： ctime、atime、mtime、cmin、amin、mmin  按时间搜索

	-exec： 将搜索结果，统一执行某一命令

		语法： find 目录  参数 关键字  -exec   待执行命令  {} \;
				------------------------
		find ./ -maxdepth 1 -type "f" -exec  ls -lh {} \;

	-xargs:  将搜索结果，统一执行某一命令

		语法：  find 目录  参数 关键字 | xargs 待执行命令

		防置搜索到的目标中，含有特殊字符（如：空格） ，使用 -print0 来弥补。： find 目录  参数 关键字 -print0 | xargs -0 待执行命令

		find ./ -maxdepth 2 -type "f" -print0 | xargs -0 ls -lh

grep： --- 按文件内容搜。 【重点】

	语法： grep -r 搜索关键字 目录

		grep -r “致爱丽丝”./ -n (显示关键字在文件中的行号)	

find、xargs、grep 结合；

	find ./ -type "f" -print0 | xargs -0 grep -r "love" -n 

	find ./ -type "f" -exec  grep -r "love" -n {} \;

vim/vi 编辑器：  ――  文本编辑器。  vim 文件名

	命令模式： i：向光标前插入， 切换工作模式  【重点】

		a：向光标后插入， 切换工作模式

		o：向关标所在行的下一行插入，切换工作模式 【重点】

		O：向关标所在行的上一行插入，切换工作模式

		I：插入光标所在行的行首。切换工作模式

		A：插入光标所在行的行尾。切换工作模式

		s：以删除一个字符为条件，切换工作模式

		S：以删除光标所在行为条件，切换工作模式

	编辑模式： 写、删除、修改文本内容

	末行模式： “：”   末行命令

移动：
	h、j、k、l：左、下、上、右

删除、剪切、粘贴：

	x：删除一个光标选中字符

	dd： 删除(剪切)光标所在行

	ndd： n表示要删除（剪切）行数。 将光标挪移至首行上。

	dw：删除一个单词。将光标置于单词首字母。

	D/d$：从光标所在位置删除（剪切）到本行行尾。

	d0：从光标所在位置删除（剪切）到本行行首。

	区域删除（剪切）：将光标挪至待剪切的首字符上，按“v”（左下角出现“可视”提示）。使用hjkl挪移光标，选择操作区域。 按d剪切。 

	p（小写）：粘贴。  向光标所在行后粘贴。

	P（大写）：粘贴。  向光标所在行前粘贴。

	yy：复制光标所在行

查找：
	“/”: 键入用户想象的待查询字符串。 n ，检索下一个。

	“*”：将光标放置于待查询单词任意字符上，使用*向后检索。使用“#”向前检索。

替换：
	“r”：单字符替换。 光标选中待替换字符，按“r”，输入要替换为的字符。

	替换一行中一个单词：

		末行下，:s/旧字符串/新字符串  ―― 替换光标所在行的第一个。 要替换全部 加：/g

	通篇替换：
		末行下，:%s/旧字符串/新字符串  ―― 替换本文所有行的第一。 要替换全部 加：/g

	指定起始、终止行替换：

		末行下，:起始行号，终止行号s/旧字符串/新字符串  ―― 替换选定行号的每行的第一个。 要替换全部 加：/g
